# Lab 05：做一个配置加载器并处理错误

## 实验目标

练习文件读写、JSON 解析、异常处理和最小日志。

## 运行方式

1. 新建一个 `config.json`
2. 写一个 `load_config()` 函数
3. 在文件缺失、JSON 非法时给出清晰错误
4. 用日志或打印记录加载步骤

## 关键代码

- 使用 `pathlib.Path`
- 使用 `json.loads()` 或 `json.load()`
- 使用 `try / except`
- 至少记录一次成功加载和一次失败信息

## 预期输出

- 配置存在时能正常读取
- 配置缺失时提示明确
- JSON 格式错误时能看出错误来源

## 常见报错

- 路径写死
- 捕获了异常但不说原因
- 读取失败时没有上下文信息

## 实验结论

做完后你应该能解释：

- 为什么错误处理是为了定位问题，不是为了吞错
- 为什么配置加载和主流程应该分开
