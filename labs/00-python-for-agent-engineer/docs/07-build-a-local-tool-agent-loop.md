# Lab 07：做一个本地工具 Agent Loop 原型

## 实验目标

把函数、数据结构、异常处理、日志和最小控制流收束成一个简单 agent 原型。

## 运行方式

写一个带 1 到 2 个本地工具的程序，例如：

- `search_tool(query: str)`
- `summarize_tool(text: str)`

然后实现：

1. 读取输入
2. 决定调用哪个工具
3. 记录每一步
4. 输出最终结果
5. 设置最大轮次或明确停止条件

## 关键代码

- `tools.py`
- `agent.py`
- `state.py` 或最小状态结构
- `main.py`

## 预期输出

- 能根据输入调用正确工具
- 中间步骤可见
- 错误输入不会直接把程序打崩
- 最终结果和中间状态分得清楚

## 常见报错

- 工具逻辑和调度逻辑混在一起
- 没有停止条件
- 只记录最终输出，不记录中间步骤
