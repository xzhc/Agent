# Agent Engineer Learning Repo

这个仓库用于从零开始学习 Agent 开发，并逐步成长为具备实际工作能力的 Agent Engineer。

建议先阅读学习流程文档：[notes/learning-workflow.md](/Users/xzh/Tech/CS/Agent/notes/learning-workflow.md)

## 目录结构

- `lectures/`：系统化学习讲义
- `labs/`：单点实验与最小练习
- `projects/`：阶段性实战项目
- `notes/`：概念笔记、术语表和踩坑记录
- `reviews/`：阶段复盘、代码审查和项目总结
- `resources/`：外部资料索引与阅读路线

## 建议使用方式

1. 先读 `lectures/` 中当前阶段的讲义。
2. 在 `labs/` 中完成单点实验，验证自己真的理解了关键机制。
3. 把讲义和实验中的能力组合起来，完成 `projects/` 中的实战项目。
4. 将过程中的疑问、错误和新理解沉淀到 `notes/`。
5. 阶段结束后在 `reviews/` 中做复盘，明确下一步提升点。
6. 如果需要补充一手资料或开源案例，到 `resources/` 查对应索引。

## 命名建议

- `lectures/`：`00-主题名/`
- `labs/`：`lab-01-主题名/`
- `projects/`：`project-01-主题名/`
- `reviews/`：`YYYY-MM-DD-主题.md`
